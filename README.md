# Seminário: Introdução à Programação com Dados Geoespaciais

Este repositório contém o materiais  que foram desenvolvidos para o seminário da disciplina de **Introdução à Programação com Dados Geoespaciais**. O foco principal é demonstrar o uso do **PyQGIS** (a API Python nativa do QGIS) integrado ao ambiente do **Jupyter Notebook**, aplicando técnicas de automação para processamento de dados vetoriais e matriciais.

## 📌 Apresentação do Projeto

O  trabalho foi projetado para introduzir os conceitos básicos do PyQGIS. Mostramos na prática como automatizar tarefas cotidianas de geoprocessamento, acessar tabelas de atributos e estruturar scripts simples diretamente no ecossistema do QGIS.

*   **Execução Modular:** Permite rodar o código célula por célula, o que facilita o ajuste fino de parâmetros e a visualização imediata de resultados parciais.
*   **Documentação Rica:** O uso de células em *Markdown* permite organizar o fluxo de trabalho com textos explicativos e documentação detalhada. Isso transforma processos complexos de geoprocessamento em um roteiro dinâmico, reprodutível e muito mais fácil de entender.

## 🎯 Objetivos Práticos

Demonstrar a aplicação da API nativa do PyQGIS dentro do ambiente do Jupyter Notebook para a realização de fluxos de trabalho geoespaciais completos, incluindo:

1.  **Manipulação Vetorial:** Criação automatizada de camadas.
2.  **Processamento de Imagens:** Realce de contraste em imagens de satélite.
3.  **Estilização Automatizada:** Aplicação de estilos e renderização de dados diretamente na interface do QGIS, sem a necessidade de cliques manuais ou do uso de bibliotecas externas de matrizes.

## 🛠️ Requisitos e Ferramentas

*   **QGIS 3.x** com suporte a Python 3.
*   **PyQGIS API** (nativa do sistema).
*   **Plugin Jupyter Notebook para QGIS** (necessário para a integração e execução do arquivo `.ipynb`).

## 📁 Como Utilizar este Repositório

1.  Certifique-se de ter o QGIS instalado e o plugin do Jupyter devidamente configurado em sua máquina.
2.  Clone este repositório ou baixe o arquivo do notebook.
3.  Abra a interface do Jupyter dentro do ambiente do QGIS.
4.  Execute as células sequencialmente para acompanhar o fluxo do seminário e visualizar as alterações acontecendo em tempo real no mapa do QGIS.
