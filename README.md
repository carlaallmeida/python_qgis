# Seminário: Introdução à Programação com Dados Geoespaciais

Este repositório  tem o objetivo de trazer projetos relacionados a disciplina de **Introdução à Programação com Dados Geoespaciais**. 

Já na pasta Seminário encontrar um projeto que tem o objetivo de mostrar como é feito um co-registro de imagens com o uso dos satélites CBERS_4A_MUX e Sentinel-2

## 📌 Seminário

Nesta pasta, você encontrará um projeto prático que demonstra como integrar o PyQGIS — a API Python nativa do QGIS — ao ambiente dinâmico do Jupyter Notebook. O material ilustra o uso de técnicas de automação voltadas ao processamento eficiente de dados vetoriais e matriciais

## 📌 Projeto

Neste pasta, você encontraráum projeto prático que aborda o desafio do alinhamento geométrico de imagens de satélite utilizando o pacote Python AROSICS.

## 🛠️ Requisitos e Ferramentas

*   **Plugin Notebook** Instalação no Qgis.
*   **Pacote  AROSICS** (nativa do sistema).
*   **Qgis** Instalação da Ferramenta.
